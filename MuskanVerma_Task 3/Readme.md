# CSS Selectors Task

This project demonstrates the use of CSS selectors such as:

- Element Selector
- Class Selector
- ID Selector
- nth-child Selector

The webpage is styled using an external CSS file.

---

## Files Included

- index.html
- style.css
- README.md

---

## Features

- Background color applied to body
- Headings styled using class selector
- Background styling using ID selector
- Paragraph styling using element selector
- Link styling without inline CSS
- Ordered list items styled using nth-child selector

---

## Technologies Used

- HTML5
- CSS3
