# Laundry Mart Website

A simple and responsive Laundry Service website created using HTML.  
This project showcases laundry services, pricing details, and a booking/contact form for customers.

## Features
- Welcome section
- Laundry service list
- Price table
- Booking form
- Simple and clean UI

## Technologies Used
- HTML5

## Services Included
- Washing and Drying
- Ironing and Folding
- Dry Cleaning
- Express Service
- Special Garment Care

## Price List
| Service | Price |
|---------|--------|
| Washing and Drying | $10 |
| Ironing and Folding | $5 |

## How to Run
1. Download or clone the project.
2. Open the project folder.
3. Double-click `index.html`.
4. The website will open in your browser.
