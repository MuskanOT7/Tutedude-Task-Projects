# Task 4 - CSS Units, Box Model & Fonts

This project is created using HTML and CSS as part of Task 4 practice.

## 📌 Features

- Proper HTML document structure
- External CSS file linked
- Responsive spacing using viewport units (`vh` and `vw`)
- Gradient background styling
- Image display inside a container
- Styled button for interaction
- Proper Box Sizing used
- No scrolling on webpage
- No Flexbox used
- No `overflow: hidden`
- No `position: relative` or `absolute`
